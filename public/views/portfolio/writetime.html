Writeti.me

  • Writeti.me is a SEAN stack app built to help establish good habits in writing daily through a clean UX experience that allows for daily personal posts.
  • Languages: JavaScript, HTML5, CSS3, AngularJS
  • Frameworks: Node.js, ExpressJS
  • Database Tools: SQL, PostgreSQL, Sequelize
  • Other: Express-JWT, Bootstrap
  • Role: Express-JWT, Bootstrap
