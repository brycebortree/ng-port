writetime

  • Writeti.me is a project I built with a SQL database, HTML, CSS, JavaScript, and AngularJS. It's a passion project of mine seeking a new text based platform online that doesn't charge a fee to simply head somewhere and spend time writing. My main focuses for this project were to incorporate a clean and attractive design, and to implement user logins.
  • Languages: JavaScript, HTML5, CSS3, AngularJS
  • Frameworks: Bootstrap
  • Server Tools: Node.js, ExpressJS
  • Database Tools: SQL, PostgreSQL, Sequelize
  • Other: Express-JWT
