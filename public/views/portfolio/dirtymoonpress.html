Dirty Moon Press
  • Dirty Moon Press is a template website for small presses, built in response to many Squarespace websites with built-in functionality that could be more catered to the individual press. I saw this as a chance to learn Angular Material, Google's front-end framework for creating Single Page Applications with Angular.
  • Languages: Javascript, JQuery
  • Frameworks: Angular, Material
  • Other: AWS, Stripe
